Second iteration

The coffee machine is not free anymore! One tea is 0,4 euro, a coffee is 0,6 euro, a chocolate is 0,5 euro.

The coffee machine only accepts coins of 10, 20, 50 cents and coins of 1 and 2 euro. The coffee machine has to give back the change to the user.

Use cases

  • I want to be able to buy a drink using euro and get the change back
  • If I do not provide enough money, I want it all back and need to see a message saying how much is missing.

Implementation details

To send a message indicating the missing amount of change you can use the drink maker protocol.

Giving back the exact change is handled by the machine. All you have to do is send the correct message to the user indicating the missing amount.
