First iteration

In this iteration, your job is to implement the engine that receives order from the user and that dispatches the command to the drink maker according to drink maker protocol.

The coffee machine can serves 3 type of drinks: tea, coffee, chocolate.

Use cases

Your product owner has delivered the stories and here they are:

  • I want to have a drink served when I pressed the corresponding drink button
  • I want to be able to add one or two sugars
  • When I have some sugar in my drink I want a stick (touillette) with it

Implementation details

The implementation of the user order is up to you. For instance, it could be a bean that contains the order details of the user.
